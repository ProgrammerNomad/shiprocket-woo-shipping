== Description ==

This plugin simplifies shipping for your WooCommerce store by integrating with Shiprocket, India's leading shipping solution. It provides accurate shipping rates, pincode serviceability checks, and a smooth checkout experience for your customers.

**Key Features:**

* **Real-time Shipping Rates:** Get live shipping rates from Shiprocket based on the customer's delivery pincode and the product's weight.
* **Dynamic Shipping Methods:** Display available shipping options to customers on the checkout page, updating dynamically as they change their pincode.
* **Pincode Serviceability Check:** Allow customers to check if their pincode is serviceable by Shiprocket directly on the product page.
* **Remember Customer Pincode:** Store the customer's pincode using localStorage for a seamless experience across product pages.
* **Custom Messages:** Display customized messages based on quick delivery options and estimated delivery times.
* **Secure Token Generation:** Automatically generate and securely store your Shiprocket API token.
* **Customizable Styles:** Apply custom CSS to the cart and checkout pages.

**Benefits:**

* **Improved Accuracy:** Ensure accurate shipping costs for every order.
* **Enhanced Customer Experience:** Provide real-time shipping information and a streamlined checkout process.
* **Increased Efficiency:** Automate shipping rate calculations and reduce manual effort.
* **Seamless Integration:** Integrate seamlessly with your WooCommerce store and Shiprocket account.

== Installation ==

1. Upload the `woo-shiprocket-shipping` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings (WooCommerce > Settings > Shipping > Shiprocket) by entering your Shiprocket email and password.

== Frequently Asked Questions ==

= How do I get a Shiprocket API token? =
You can obtain your Shiprocket API token by logging into your Shiprocket account and navigating to the API settings section.

= What if my pincode is not serviceable? =
If a customer's pincode is not serviceable, they will see a message indicating that, along with any potential reasons or suggestions.
